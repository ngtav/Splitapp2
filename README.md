# 🍽️ SplitEat — Dividí la cuenta

App mobile-first para distribuir gastos entre amigos, con soporte para dietas, alcohol, propinas, eventos compartidos en tiempo real y exportación a Excel e imagen.

## ✨ Features

- 👥 **Amigos** con dieta (veggie/carne), alcohol y alias CBU/CVU
- 🧾 **Gastos** por tipo: todos, carnívoros, vegetarianos, alcohol, sin alcohol, propina/bono
- 💰 **Resultado automático**: quién debe, quién cobra, transferencias exactas
- 📊 **Exportar a Excel**: 3 hojas (resumen, matriz de pagos, detalle de gastos)
- 🖼️ **Exportar como imagen**: matriz visual lista para compartir
- 💬 **Compartir por WhatsApp**: resultado formateado con alias incluidos
- 📅 **Clonar eventos**: reutilizá el mismo grupo para la próxima salida
